[![codecov](https://codecov.io/gh/infra-blocks/ts-toolkit/graph/badge.svg?token=QQ3Q61NY0D)](https://codecov.io/gh/infra-blocks/ts-toolkit)

This repository stores the code of the [@infra-blocks/toolkit](https://www.npmjs.com/package/@infra-blocks/toolkit) package. The package aims to
provide low-level utilities aimed at maximizing code reuse across the organization. It is inspired by, and aims to be a complement of, many similar
libraries, such as [es-toolkit](https://es-toolkit.dev/).
